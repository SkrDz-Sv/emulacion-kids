Emulacion-kids

Portal o plataforma para iniciar juegos de nes, snes y nintendo 64. Utilizando el joystic como mouse y dirigido a niños mayores de 4 años.

Introducción a Emulacion-Kids

Pretendemos crear un programa que se pueda inicializar al cargar el sistema operativo UBUNTU (inicialmente claro esta...) y ser una plataforma sencilla para poder elegir el emulador a utilizar (para NES, SNES o N64) y luego elegir el juego, todo desde el joystick.

Alcance

Obviamente, por la simplicidad y el tipo de juegos (ROMS) que se recomendarían serían enfocados a niños mayores de 4 años.

Objetivo final

El objetivo es darle una mini laptop (por ejemplo) a un niño y dejar todo listo de tal manera que al encender la mini laptop se cargue el programa y con el joystick se pueda elegir el emulador, el juego y a jugar! Todo sencillo. Deberá de tener alguna forma sencilla de poder cerrar el emulador y poder estar en la aplicación nuevamente para elegir otro emulador y jugar. Al final, existirá una opción para cerrar la aplicación fácilmente y apagar automáticamente el SO. ¿Una nueva consola? Podría ser... pero un poco más artesanal y enfocado a niñ@s!. Al final, lo que queremos es que no tengan acceso directo a la laptop, sino solo a un switch de encendido (por así decirlo) y que éste programa se encargue de todo.
